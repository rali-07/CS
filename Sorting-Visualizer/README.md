🧮 排序算法可视化 Sorting Visualizer

这是一个使用 Python Tkinter 编写的排序算法可视化工具，通过柱状图动态展示各种排序算法的执行过程。支持随机数据生成、自定义数据导入、速度调节、暂停/继续等功能。

✨ 特性

· 📊 实时柱状图：无闪烁绘制，数据变化平滑
· 🎚 可调节速度：滑动条或手动输入
· 🎲 随机数据生成：自动生成 1~n 的随机排列
· 📂 自定义数据导入：支持逗号或空格分隔的数字
· 🧠 多种排序算法：
  · 冒泡排序
  · 归并排序
  · 选择排序
  · 插入排序
  · 快速排序
  · 堆排序
  · 希尔排序
  · 猴子排序（娱乐版）
· ⏯ 暂停/继续：可在排序过程中暂停，随时继续
· 📝 日志系统：记录每一步关键操作和时间戳
· 🎨 深色主题：黑色背景 + 亮色柱子，保护眼睛

🚀 如何运行

环境要求

· Python 3.6 或更高版本
· 不需要额外安装第三方库（Tkinter 和 random, re, time 等均为标准库）

运行步骤

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/rali-07/CS.git
   cd CS/Sorting-Visualizer
   ```
   
2. 运行主程序：
   ```bash
   python sort_visualizer.py
   ```

🎮 使用说明

1. 生成数据：在右侧“数据数量”输入数字，点击“生成随机数据”。
2. 导入自定义数据：在输入框输入数字（如 5,12,8,20,15），点击“导入数据”。
3. 选择算法：从下拉框选择你想看的排序算法。
4. 调整速度：拖动滑动条或直接输入毫秒数（1~1000）。
5. 开始排序：点击“开始排序”，动画开始。
6. 暂停/继续：点击“停止排序”可暂停，再次点击“开始排序”继续。
7. 切换算法/修改数据：会自动重置排序状态，不用担心冲突。

🛠 技术实现

· 无闪烁绘图：使用对象缓存，只更新坐标和颜色，不删除全部重绘。
· 生成器 + after 循环：将排序算法拆解为可控步骤，不阻塞 UI。
· 高亮系统：通过 yield 返回需要高亮的柱子索引，灵活支持多种算法。
· 日志系统：带时间戳的颜色日志，便于跟踪。
